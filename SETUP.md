# Discord Clone - Инструкция по установке и запуску

## Требования
- Node.js v14 или выше
- npm или yarn
- Современный браузер (Chrome, Firefox, Edge)

## Быстрый старт

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск сервера
```bash
npm start
```

Сервер запустится на `http://localhost:3000`

**Телефон в той же Wi‑Fi сети:** по обычному `http://IP:3000` браузер **не даст** микрофон, камеру и звонки (нет *secure context*). В `.env` на ПК с сервером задайте `USE_HTTPS=true` и `FLOR_TLS_SAN=localhost,127.0.0.1,IP_вашего_ПК`, перезапустите `npm start` и на телефоне откройте `https://IP:3000/login.html` (один раз примите предупреждение о сертификате). На экране входа при HTTP также показывается красная подсказка.

### 3. Открытие приложения

**В браузере:**
Откройте `http://localhost:3000/login.html`

**Десктоп приложение:**
В отдельном терминале:
```bash
npm run electron
```

## Первый запуск

1. **Регистрация**
   - Откройте приложение
   - Нажмите "Register"
   - Введите:
     - Имя пользователя (минимум 3 символа)
     - Email
     - Пароль (минимум 6 символов)
   - Подтвердите пароль
   - Нажмите "Register"

2. **Вход**
   - Введите email и пароль
   - Нажмите "Log In"

## Основные функции

### Друзья
1. Нажмите на иконку "Friends" (первая иконка в списке серверов)
2. Перейдите на вкладку "Add Friend"
3. Найдите пользователей и отправьте запрос
4. Принимайте запросы во вкладке "Pending"
5. Общайтесь с друзьями во вкладке "Online" или "All"

### Сообщения
1. Выберите канал в списке слева
2. Введите сообщение в поле внизу
3. Нажмите Enter для отправки

### Загрузка файлов
1. Нажмите на кнопку скрепки (📎)
2. Выберите файл (до 10MB)
3. Файл появится в чате

### Эмодзи реакции
1. Наведите на сообщение
2. Нажмите на кнопку с эмодзи
3. Выберите эмодзи из списка

### Голосовые/Видео звонки
1. Нажмите на голосовой канал
2. Разрешите доступ к камере и микрофону
3. Используйте кнопки для:
   - Включения/выключения камеры
   - Включения/выключения микрофона
   - Демонстрации экрана
4. Двойной клик на видео для развертывания

### Создание сервера
1. Нажмите на кнопку "+" в списке серверов
2. Введите название сервера
3. Сервер появится в списке

## Решение проблем

### Проблема с входом/редиректом
Если приложение постоянно переключается между страницами:
1. Очистите localStorage браузера
2. Перезапустите сервер
3. Зарегистрируйтесь заново

### Камера/микрофон не работают
1. С телефона по `http://IP` — включите HTTPS (см. раздел «Телефон» выше).
2. Проверьте разрешения браузера
3. Убедитесь, что другие приложения не используют устройства
4. Обновите страницу

### Ошибка загрузки файлов
1. Проверьте размер файла (максимум 10MB)
2. Убедитесь, что папка `uploads/` создана
3. Проверьте права доступа к папке

### База данных
- Файл базы данных: `discord_clone.db`
- Для сброса: удалите файл и перезапустите сервер
- Новая база создастся автоматически

## Очистка данных

Для полной очистки:
```bash
# Удалить базу данных
rm discord_clone.db

# Очистить загруженные файлы
rm -rf uploads/*

# Очистить node_modules (опционально)
rm -rf node_modules
npm install
```

## Порты
- Веб-сервер: 3000
- WebSocket: автоматически (Socket.IO)
- WebRTC: P2P соединения (порты динамические)

## Безопасность
- Пароли хешируются с помощью bcrypt
- JWT токены для аутентификации
- Сессии хранятся в localStorage
- HTTPS требуется для production

## Оболочка «только сайт» (EXE без сервера внутри)

Окно, которое открывает уже развёрнутый в интернете или в LAN FLOR (как отдельное приложение).

1. В `.env` можно задать `FLOR_WEB_URL=https://ваш-сервер/login.html` (или передать URL аргументом).
2. Для самоподписанного HTTPS в LAN: `ELECTRON_ALLOW_SELF_SIGNED=true`.
3. Сборка:

```bash
npm run build:shell
```

Артефакты — в папке `dist-shell/` (portable exe и установщик NSIS).

Проверка без сборки:

```bash
npm run electron:shell
```

## Android APK (WebView на тот же сайт)

1. Установите [Android Studio](https://developer.android.com/studio) и JDK.
2. В корне проекта отредактируйте `capacitor.config.json` → `server.url` на реальный адрес сервера (для звонков лучше **https://**). Значение `https://127.0.0.1:3000` на реальном телефоне указывает на сам телефон — замените на IP/домен ПК или VPS.
3. Команды:

```bash
npm install
npx cap add android
npm run cap:sync
npm run cap:open:android
```

В Android Studio: **Build → Build Bundle(s) / APK(s) → Build APK(s)**. Папка `android/` создаётся локально и по умолчанию в `.gitignore`; при необходимости закоммитьте её сами.
